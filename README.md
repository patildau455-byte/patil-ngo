# patil-ngo
PATIL NGO project features a public website using HTML/Tailwind CSS. The backend dashboard uses PHP/MySQL for admin control (login, banner management). Database setup is included.
 PATIL NGO Website & Admin Dashboard
This project is a foundational development for the PATIL NGO, consisting of a public-facing website built with HTML and Tailwind CSS for a modern, responsive design, and a backend Admin Dashboard developed with PHP and MySQL for content management.

 Key Features
Public Frontend (frontend/index.html, frontend/styles.css):

Modern Layout: Uses Tailwind CSS for utility-first styling, ensuring a clean and responsive user interface.

Core Pages: Includes a basic navigation structure for Home, About, Programs, Donate, and Contact.

Dynamic Banner: Features a prominent header banner with a compelling call-to-action text, "Empowering Communities, Uplifting Lives."

Backend & Administration (backend/ directory):

Admin Dashboard (backend/dashboard.php): The central hub for administrators to manage website content.

Content Management (WIP): Includes a dedicated page for managing the website's banner (backend/manage_banner.php), suggesting future content control features.

Session-Based Login (backend/login.php): Implements a basic login system using PHP sessions for secure administrator access.

Database Integration:

MySQL Setup (database/patil_ngo.sql): Provides the necessary SQL command to initialize the patil_ngo database.

Configuration (backend/config.php): Contains the PHP connection logic to connect the application to the MySQL database (localhost, user root, no password).

 Technology Stack
Frontend: HTML, CSS, Tailwind CSS, JavaScript (Implicit, for future use)

Backend: PHP (with Session Management)

Database: MySQL
